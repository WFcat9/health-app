## 🚀 如何使用

### 第一步：访问登录页面
打开 `login-final.html`

### 第二步：注册账号
1. 点击"注册 📝"标签
2. 填写昵称、邮箱、密码
3. 点击"注册 🎉"按钮

### 第三步：登录系统
1. 注册成功后会自动跳转，或手动切换到登录标签
2. 输入邮箱和密码
3. 点击"登录 🚀"按钮

### 第四步：使用功能
登录后会进入主页面（main-final.html），你可以：
- 📋 查看健康康复计划
- ✅ 进行习惯打卡
- 📊 记录健康数据
- 🎬 观看健康视频

## 📝 技术说明

### 数据存储
- 使用localStorage存储用户数据和健康数据
- 用户数据键名：`healthAppUsers`
- 当前用户键名：`healthAppCurrentUser`
- 健康数据键名：`healthAppData_${userId}`

### 视频功能
- 使用YouTube嵌入视频
- 视频数据包含真实的YouTube视频ID
- 点击视频卡片即可播放
- 点击×按钮或按ESC键可以关闭视频

### 响应式设计
- 所有页面都支持移动端访问
- 使用CSS Grid和Flexbox布局
- 在768px宽度下有专门的移动端样式
